# Meteor Client Archive

This repository serves as an archive for Meteor Client builds and **nothing else**.

## Purpose

This archive automatically collects and stores Meteor Client builds organized by Minecraft version.

## Disclaimer

**WARNING: These versions may be archived and PROBABLY ARE NOT supported!**

- The builds stored here may be **old** and outdated
- You will **NOT receive any help** or support when using these versions from any of Meteor developers, so do not bother them. 
- This repository is for archival purposes only.
- For the latest supported version, visit the [official Meteor Client website](https://meteorclient.com)

For more information about old versions, see the [official Meteor Client FAQ on old versions](https://meteorclient.com/faq/old-versions).

## Usage

If you choose to use builds from this archive:

1. **You are on your own** - no support will be provided
2. These are completely old versions **and no longer maintained** by Meteor Development. May contain exploits, bugs, crashes etc.  
3. **Always prefer** the latest version from the official source
